# 資策會MFEE07大專題`（前端 React.js）`

## 第四組 香氛電商平台

+ 基礎技術: HTML5+ sass<br />
+ 前端技術: React.js + Redux + JavaScript(ES6)<br /> 
+ 後端技術: Node.js + express.js + MySQL <br />
+ 前端套件: MATERIAL-UI<br />
+ 版本控制: git (github)<br />
+ 編輯器技術: vscode<br />
